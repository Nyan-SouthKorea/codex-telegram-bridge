# Agent Guide

이 레포에서 작업할 때는 먼저 [docs/agent.md](docs/agent.md)를 읽습니다.

최소 규칙:

- 작업 전 [docs/README.md](docs/README.md)와 [docs/status.md](docs/status.md)를 먼저 확인합니다.
- 현재 상태는 `docs/status.md`를 단일 기준 문서로 유지합니다.
- 같은 사실을 여러 문서에 중복으로 누적하지 않습니다.
- 레포 안에 둘 수 있는 서비스/스크립트/설정 템플릿은 레포 안에 저장합니다.
- 실제 비밀값과 개인 정보는 커밋하지 않습니다.
